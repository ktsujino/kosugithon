API description
- set
ユーザがカフェにきたとき http://210.140.71.170:8000/set?username=shimizu&atcafe=True
ユーザがカフェからいなくなったとき http://210.140.71.170:8000/set?username=shimizu&atcafe=False
ユーザを登録するとき・ユーザが教えたいことや教わりたいことを変更するとき
例1 おそわる: http://210.140.71.170:8000/set?username=tsujino&description=トランペットの吹き方&teach=False
例2 おしえる: http://210.140.71.170:8000/set?username=shimizu&atcafe=False&description=プロジェクトマネジメント入門&teach=True

- get
例: http://210.140.71.170:8000/set?username=shimizu
知らないユーザについては以下だけを返す
http://210.140.71.170:8000/get?username=foo -> {"username" : "foo", "atcafe" : "False"}
知ってるユーザについては以下を返す
http://210.140.71.170:8000/get?username=shimizu -> {"username" : "shimizu", "atcafe" : "False", "description": "プロジェクトマネジメント入門", "teach" : "True"}

- get_people
引数なし。
http://210.140.71.170:8000/get_people -> {"teachers" : "[shimizu]", "students" : "[tsujino]"}
